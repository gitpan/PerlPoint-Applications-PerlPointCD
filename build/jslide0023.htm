Welcome and Impressum

To complete the CD, one can add a welcome text and an impressum, both written in PerlPoint or POD as usual.

The makefile has macros to add these parts:

    The INTROFILE macro integrates an optional startup part.
    The IMPRESSUM macro adds an optional postambel.

They can be used like this: 


  make INTROFILE=prepare/welcome.pp \
       IMPRESSUM=prepare/impressum.pp

The prepare directory used in this example is the recommended place for such sources, according to the directory structure, but you are free to choose your own.
