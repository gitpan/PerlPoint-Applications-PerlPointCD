The master CD

We have the talk materials, welcome text and impressum, we have layouts and a startup page, and of course a passed deadline, so ... the CD is ready! Now we can go for a master CD.

The way to do this is almost the same as for every intermediate version. Just add an additional target cd to the call of make: 


  make cd

Additionally to the usual steps of making HTML pages, directories are renamed to a DOS, ISO and web address compliant scheme, and the pages are generated without validation links.

As usual, the results can be found in the build directory. To burn the CD, just burn the contents of this directory (make it the CD root dir).
