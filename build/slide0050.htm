Supports speaker teams

Whether a talk is given by exactly one speaker or several persons together, PerlPointCD can handle it:

    The author level in the directory structure can hold multiple names, separated by commas and (optional) underscores (or spaces). 


  First_Author,_Second_Author_,_Third_Author

    If author defaults shall be stored, there should be one directory for each of the authors. 


  defaults/authors/First_Author
  defaults/authors/Second_Author
  defaults/authors/Third_Author

Entries in the author index will be made for each of those authors. In general, the tool handles each of the authors individually.
