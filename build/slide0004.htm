01: Installation

How to get this package and make it run? Well, installing PerlPointCD is as simple as the installation of any CPAN package:

    Install the prerequisites: PerlPoint::Package, PerlPoint::Converters and Pod::PerlPoint. Alternatively, users of CPAN and CPANPLUS might find Bundle::PerlPoint helpful to get them all at once.
    Get the PerlPointCD package. It is stored on CPAN (PerlPoint::Applications::PerlPointCD) and SourceForge (see www.sourceforge.net/projects/perlpoint).
    Unpack the package: 


  gtar zxvf PerlPoint-Applications-PerlPointCD-<version>.tgz

This makes a new directory PerlPoint-Applications-PerlPointCD-<version>. I can switch therein and start the CD project! This is my working directory.
