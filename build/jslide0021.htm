Startup page

Having several layout variants requires to have a startup page that offers the choices. For the German Perl Workshop, we used an index.html file in the build directory (which becomes the CD startup directory later on). A derived copy of this page is provided with this demo, so please have a look. This page is handcrafted, so one can imagine all kinds of solutions. Our page tried to detect whether the user has Java activated or not, and then tries to redirect automatically to the layout that is probably preferred - after a period of 60 seconds or so.
