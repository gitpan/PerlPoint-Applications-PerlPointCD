Validation support

Generated pages prior the final make call for CD mastering contain links to validate a page by public services. There are links for HTML validation, CSS validation and link checks.

Please note that due to the principles of site checking used by these services, validation links only work if the CD is placed on a webserver.

In the final make call (with target cd), these validation links are suppressed.
