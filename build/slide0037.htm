Extraordinary long titles can be easily shortened in chapter links

PerlPoint layouts allow to use navigation references, like in this demo: each page has a navigation bar with links to the next and the previous page.

If a title of a talk is very long, the resulting navigation bars can look a little bit misdesigned. In this case one can declare a short title version by adding a talk.short file to the talk directory (where talk.pp is located). This file should contain the short title version in its first line. 


  For an example of the effect, look at the
  navigation links to this page - in the
  navigation bars of the previous and following
  pages.

This feature works for all levels of the directory structure - files need to be named level specific like style.short, author.short etc. - but most likely the talk level is the one where it is mostly required.
